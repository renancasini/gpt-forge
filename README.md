# gpt-forge

Base mínima e reutilizável para criação de projetos modernos.

Este repositório não é um projeto final.
É uma fundação: padrões, estrutura e ferramentas para iniciar e manter
outros repositórios de forma consistente.

## O que é

- Um ponto de partida único
- Um conjunto de convenções
- Templates prontos para uso
- Infraestrutura base (Docker, scripts, docs)

## O que não é

- Não é um app
- Não é um framework fechado
- Não é um projeto específico

## Como usar

1. Clone este repositório
2. Use os templates como base
3. Crie um novo repositório para cada projeto
4. Ajuste apenas o necessário
